Premiers pas avec Python et la fonction print()

Objectifs d'apprentissage

  1. Comprendre la fonction print() et ses paramètres
  2. Maîtriser différentes façons d'afficher du texte
  3. Utiliser les caractères spéciaux et sauts de ligne
  4. Explorer les options avancées de print()

1. Les bases de la fonction print()

La fonction print() est l'une des fonctions les plus fondamentales en Python. Elle permet d'afficher du texte et d'autres données à l'écran.

Affichage simple :

print("Marhaba!") # Affiche: Marhaba!

Affichage de plusieurs éléments :

print("Bienvenue au", "Maroc") # Les éléments sont séparés par des espaces

Affichage avec différents types de données :

print("La température à Marrakech:", 25, "degrés") # Mélange texte et nombres

Résultat après exécution :

Marhaba!
Bienvenue au Maroc
La température à Marrakech: 25 degrés

2. Paramètres de la fonction print()

Paramètre 'sep' : séparateur personnalisé

Le paramètre sep permet de définir le séparateur entre les éléments affichés :

print("Casablanca", "Rabat", "Fès", sep=" ➔ ")
print("Maroc", "Maghreb", "Afrique", sep=" | ")

Paramètre 'end' : fin de ligne personnalisée

Le paramètre end permet de définir ce qui sera affiché à la fin de la ligne :

print("Le Maroc est", end=" ")
print("un beau pays") # Les deux print seront sur la même ligne

Résultat après exécution :

Casablanca ➔ Rabat ➔ Fès
Maroc | Maghreb | Afrique
Le Maroc est un beau pays

3. Caractères spéciaux

Python permet d'utiliser des caractères spéciaux dans les chaînes de caractères :

print("Villes du Maroc:\n- Tanger\n- Agadir\n- Meknès") # \n pour nouvelle ligne
print("Prix du tajine: 50\tDH") # \t pour tabulation
print("Le restaurant \"Dar Zwin\" est excellent") # \" pour les guillemets

Caractères spéciaux courants :

  • \n : Saut de ligne
  • \t : Tabulation
  • \" : Guillemet double
  • \' : Guillemet simple
  • \\ : Barre oblique inverse

Résultat après exécution :

Villes du Maroc:
- Tanger
- Agadir
- Meknès
Prix du tajine: 50 DH
Le restaurant "Dar Zwin" est excellent

4. Formatage avancé

Python offre plusieurs méthodes pour formater des chaînes de caractères :

F-strings (Python 3.6+) :

nom_ville = "Chefchaouen"
altitude = 564
print(f"La ville de {nom_ville} est à {altitude}m d'altitude") # f-string

Méthode format() :

print("La ville de {} est à {}m d'altitude".format(nom_ville, altitude)) # .format()

Résultat après exécution :

La ville de Chefchaouen est à 564m d'altitude
La ville de Chefchaouen est à 564m d'altitude

5. Exemples pratiques

Menu de restaurant :

print("-" * 40) # Répétition de caractère
print("Menu du Restaurant Marocain".center(40)) # Centrage
print("-" * 40)
print("1. Tajine de poulet\t\t65 DH")
print("2. Couscous royal\t\t70 DH")
print("3. Pastilla au poulet\t\t80 DH")
print("-" * 40)

Informations touristiques :

destinations = ["Marrakech", "Fès", "Meknès", "Rabat"]
print("Circuit touristique:", *destinations, sep=" -> ") # Unpacking de liste

Résultat après exécution :

----------------------------------------
Menu du Restaurant Marocain
----------------------------------------
1. Tajine de poulet 65 DH
2. Couscous royal 70 DH
3. Pastilla au poulet 80 DH
----------------------------------------
Circuit touristique: Marrakech -> Fès -> Meknès -> Rabat

6. Notes techniques importantes

  1. La fonction print() :
    • Affiche des objets sur la sortie standard
    • Convertit automatiquement les objets en chaînes
    • Ajoute par défaut un saut de ligne à la fin
  2. Paramètres principaux :
    • sep=" " : Séparateur entre les arguments (espace par défaut)
    • end="\n" : Caractère de fin (saut de ligne par défaut)
    • file=sys.stdout : Flux de sortie (sortie standard par défaut)
    • flush=False : Vidage du buffer (False par défaut)
  3. Bonnes pratiques :
    • Utiliser les f-strings pour le formatage (Python 3.6+)
    • Éviter les print() vides, préférer print("")
    • Penser à la lisibilité du texte affiché
    • Être cohérent dans le style d'affichage
