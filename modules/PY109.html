Structures conditionnelles : if, elif, else

Objectifs d'apprentissage

  1. Maîtriser les structures conditionnelles
  2. Comprendre l'imbrication des conditions
  3. Utiliser les conditions composées
  4. Appliquer les bonnes pratiques

1. Structure if simple

La structure conditionnelle la plus simple est le if qui exécute du code uniquement si une condition est vraie :

temperature = 35

if temperature > 30:
    print("Il fait chaud aujourd'hui à Marrakech!")
    print("Pensez à bien vous hydrater")

Le code indenté sous le if ne s'exécute que si la condition est vraie (True).

2. Structure if-else

La structure if-else permet d'exécuter un bloc de code si la condition est vraie, et un autre bloc si elle est fausse :

budget = 500

if budget >= 300:
    print("Vous pouvez réserver un bon hôtel")
else:
    print("Pensez plutôt à une auberge de jeunesse")

3. Structure if-elif-else

Pour gérer plusieurs conditions alternatives, on utilise la structure if-elif-else :

note = 15

if note >= 16:
    mention = "Excellent"
elif note >= 14:
    mention = "Très bien"
elif note >= 12:
    mention = "Bien"
elif note >= 10:
    mention = "Passable"
else:
    mention = "À améliorer"

print(f"Mention : {mention}")

Python teste chaque condition dans l'ordre et exécute le premier bloc dont la condition est vraie.

4. Conditions imbriquées

Il est possible d'imbriquer des structures conditionnelles les unes dans les autres :

saison = "été"
temperature = 35

if saison == "été":
    if temperature > 30:
        print("Forte chaleur, restez à l'ombre")
        if temperature > 40:
            print("Canicule! Évitez de sortir")
    else:
        print("Température normale pour l'été")
else:
    print("Ce n'est pas l'été")

L'indentation est essentielle pour définir la portée de chaque bloc conditionnel.

5. Conditions composées

Les opérateurs logiques permettent de combiner plusieurs conditions :

prix = 200
stock = 5
en_promotion = True

if prix < 300 and stock > 0:
    print("Article disponible à la vente")
    if en_promotion or stock > 10:
        print("Bonne affaire!")

Les opérateurs and, or et not permettent de créer des conditions complexes.

6. Exemple pratique - Système de réservation d'un riad

Voici un exemple complet d'utilisation des structures conditionnelles :

def verifier_reservation_riad(nombre_personnes, duree_sejour, budget_total):
    """
    Système de vérification pour la réservation d'un riad à Marrakech
    """
    # Prix par nuit selon la saison
    prix_base = 800 # en DH
    
    # Calculs préliminaires
    cout_total = prix_base * duree_sejour
    
    # Vérifications avec messages détaillés
    if nombre_personnes <= 0 or duree_sejour <= 0:
        return "Données invalides"
    
    if nombre_personnes > 6:
        return "Désolé, ce riad ne peut accueillir plus de 6 personnes"
    
    if duree_sejour < 2:
        return "La durée minimum de séjour est de 2 nuits"
        
    if budget_total < cout_total:
        message = f"""
        Budget insuffisant:
        - Coût total: {cout_total} DH
        - Votre budget: {budget_total} DH
        - Différence: {cout_total - budget_total} DH
        """
        return message
    
    # Si toutes les conditions sont remplies
    return f"""
    Réservation possible!
    - Coût total: {cout_total} DH
    - Durée: {duree_sejour} nuits
    - Personnes: {nombre_personnes}
    """

# Test du système
print("\nTest du système de réservation:")
print(verifier_reservation_riad(4, 3, 3000))
print(verifier_reservation_riad(2, 5, 2000))

Cet exemple montre comment combiner plusieurs conditions pour créer un système de vérification complet.

7. Notes techniques importantes

Voici quelques points importants à retenir :

  • Structure des conditions :
    if condition:
        # code
    elif autre_condition:
        # code
    else:
        # code
  • Opérateurs de comparaison :

    ==, !=, <, >, <=, >=

  • Opérateurs logiques :

    and, or, not

  • Bonnes pratiques :
    • Une seule condition par ligne
    • Éviter les conditions trop complexes
    • Utiliser des parenthèses pour clarifier
    • Préférer les conditions positives
