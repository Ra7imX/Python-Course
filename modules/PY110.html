La boucle while et ses cas d'utilisation

Objectifs d'apprentissage

  1. Comprendre le fonctionnement de la boucle while
  2. Maîtriser les conditions d'arrêt
  3. Éviter les boucles infinies
  4. Utiliser while dans des cas pratiques

1. Boucle while simple

La boucle while permet de répéter des instructions tant qu'une condition est vraie. Voici un exemple avec un compte à rebours :

# Compte à rebours avant le départ d'un train
temps_restant = 5

while temps_restant > 0:
    print(f"Départ dans {temps_restant} minutes...")
    temps_restant -= 1
print("Le train part maintenant!")

Résultat après exécution :

Départ dans 5 minutes...
Départ dans 4 minutes...
Départ dans 3 minutes...
Départ dans 2 minutes...
Départ dans 1 minutes...
Le train part maintenant!

2. Boucle avec condition complexe

Une boucle while peut utiliser des conditions plus élaborées :

solde = 1000
prix_souvenir = 200

while solde >= prix_souvenir and prix_souvenir > 0:
    print(f"Achat d'un souvenir. Solde restant : {solde} DH")
    solde -= prix_souvenir

Résultat après exécution :

Achat d'un souvenir. Solde restant : 1000 DH
Achat d'un souvenir. Solde restant : 800 DH
Achat d'un souvenir. Solde restant : 600 DH
Achat d'un souvenir. Solde restant : 400 DH
Achat d'un souvenir. Solde restant : 200 DH

3. Boucle avec entrée utilisateur

Les boucles while sont idéales pour gérer les entrées utilisateur :

ville_a_deviner = "Fès"
tentatives = 0
max_tentatives = 3

while tentatives < max_tentatives:
    reponse = input("Quelle est la ville la plus ancienne du Maroc ? ").strip().title()
    tentatives += 1
    
    if reponse == ville_a_deviner:
        print(f"Bravo! Vous avez trouvé en {tentatives} essais!")
        break
    else:
        essais_restants = max_tentatives - tentatives
        if essais_restants > 0:
            print(f"Essayez encore! Il vous reste {essais_restants} essais")
        else:
            print(f"Désolé, la réponse était {ville_a_deviner}")

4. Utilisation de while avec des listes

On peut parcourir des collections avec while :

destinations = ["Marrakech", "Fès", "Meknès", "Rabat", "Tanger"]
index = 0

while index < len(destinations):
    print(f"Étape {index + 1} : {destinations[index]}")
    index += 1

Résultat après exécution :

Étape 1 : Marrakech
Étape 2 : Fès
Étape 3 : Meknès
Étape 4 : Rabat
Étape 5 : Tanger

5. Exemple pratique - Simulation d'un guichet automatique

Voici un exemple plus complet d'utilisation de while pour simuler un guichet automatique :

def simuler_guichet():
    solde = 1000
    tentatives_max = 3
    tentatives = 0
    
    while tentatives < tentatives_max:
        print("\n=== Guichet Automatique ===")
        print(f"Solde actuel : {solde} DH")
        print("1. Retrait")
        print("2. Dépôt")
        print("3. Quitter")
        
        choix = input("Votre choix (1-3) : ")
        
        if choix == "1":
            montant = float(input("Montant à retirer : "))
            if montant <= solde and montant > 0:
                solde -= montant
                print(f"Retrait effectué. Nouveau solde : {solde} DH")
            else:
                print("Montant invalide ou solde insuffisant")
        
        elif choix == "2":
            montant = float(input("Montant à déposer : "))
            if montant > 0:
                solde += montant
                print(f"Dépôt effectué. Nouveau solde : {solde} DH")
            else:
                print("Montant invalide")
        
        elif choix == "3":
            print("Merci d'avoir utilisé nos services")
            break
        
        else:
            print("Choix invalide")
            tentatives += 1
            if tentatives < tentatives_max:
                print(f"Il vous reste {tentatives_max - tentatives} essais")
    
    if tentatives >= tentatives_max:
        print("Nombre maximum de tentatives atteint. Au revoir!")

6. Notes techniques importantes

  • Structure de base :
    while condition:
        # code à répéter
  • Éléments clés :
    • Condition d'arrêt claire
    • Modification de la condition dans la boucle
    • break pour sortie anticipée
    • continue pour passer à l'itération suivante
  • Pièges à éviter :
    • Boucles infinies (condition toujours vraie)
    • Conditions qui ne changent jamais
    • Oubli de l'incrémentation/décrémentation de variables
