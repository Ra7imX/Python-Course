Les opérateurs de comparaison et expressions booléennes

Objectifs d'apprentissage

  1. Comprendre les opérateurs de comparaison
  2. Maîtriser les expressions booléennes
  3. Combiner les comparaisons
  4. Utiliser les comparaisons dans des conditions

1. Opérateurs de comparaison de base

Python offre plusieurs opérateurs pour comparer des valeurs :

prix_tajine = 75
prix_couscous = 85

# Égalité (==)
print(prix_tajine == prix_couscous) # False

# Différence (!=)
print(prix_tajine != prix_couscous) # True

# Supérieur (>)
print(prix_tajine > 50) # True

# Inférieur (<)
print(prix_tajine < prix_couscous) # True

# Supérieur ou égal (>=)
print(prix_tajine >= 75) # True

# Inférieur ou égal (<=)
print(prix_couscous <= 85) # True

Chaque opérateur renvoie une valeur booléenne (True ou False) selon que la comparaison est vraie ou fausse.

2. Comparaisons de chaînes

Les opérateurs de comparaison fonctionnent également avec les chaînes de caractères :

ville1 = "Casablanca"
ville2 = "casablanca"

print(ville1 == ville2) # False (sensible à la casse)
print(ville1.lower() == ville2.lower()) # True

# Ordre alphabétique
villes = ["Rabat", "Agadir", "Fès"]
print("Agadir" < "Rabat") # True (ordre alphabétique)
print(sorted(villes)) # Trie les villes

Les comparaisons de chaînes sont basées sur l'ordre lexicographique (ordre du dictionnaire) et sont sensibles à la casse.

3. Opérateurs logiques

Python propose trois opérateurs logiques principaux pour combiner des expressions booléennes :

# AND (et)
prix = 100
stock = 5
disponible = prix <= 150 and stock > 0
print(f"Article disponible : {disponible}")

# OR (ou)
en_promotion = prix < 80 or stock > 10
print(f"En promotion : {en_promotion}")

# NOT (non)
rupture_stock = not (stock > 0)
print(f"Rupture de stock : {rupture_stock}")

Tableau des opérateurs logiques :

  • and : renvoie True si les deux conditions sont vraies
  • or : renvoie True si au moins une des conditions est vraie
  • not : inverse la valeur booléenne (True devient False et vice versa)

4. Cas pratique - Système de réservation

Exemple d'application pratique des comparaisons dans un système de réservation d'hôtel :

def verifier_disponibilite_hotel(personnes, budget, jours):
    """
    Vérifie si une réservation d'hôtel est possible
    selon les critères donnés
    """
    prix_par_jour = 300
    capacite_max = 4
    duree_min = 2
    duree_max = 15
    
    # Vérifications multiples
    prix_total = prix_par_jour * jours
    budget_suffisant = prix_total <= budget
    capacite_ok = personnes <= capacite_max
    duree_valide = duree_min <= jours <= duree_max
    
    # Combinaison des conditions
    reservation_possible = (
        budget_suffisant and
        capacite_ok and
        duree_valide
    )
    
    # Retour détaillé
    return {
        "possible": reservation_possible,
        "raisons": {
            "budget_ok": budget_suffisant,
            "capacite_ok": capacite_ok,
            "duree_ok": duree_valide
        }
    }

# Test du système
resultat = verifier_disponibilite_hotel(
    personnes=3,
    budget=1000,
    jours=4
)

print("\nRésultat de la vérification :")
print(f"Réservation possible : {resultat['possible']}")
for critere, valeur in resultat['raisons'].items():
    print(f"{critere} : {valeur}")

Dans cet exemple, nous utilisons plusieurs comparaisons combinées pour vérifier si une réservation d'hôtel est possible.
