Paramètres de fonction et valeurs par défaut

Objectifs d'apprentissage

  1. Comprendre les différents types de paramètres
  2. Maîtriser les paramètres par défaut
  3. Utiliser les arguments nommés
  4. Gérer un nombre variable de paramètres

1. Types de paramètres

Fonction avec paramètres obligatoires et optionnels :

def reserver_hotel(nom, ville, nuits=1, petit_dejeuner=True):
    """
    Fonction avec paramètres obligatoires et optionnels
    - nom, ville : paramètres obligatoires
    - nuits, petit_dejeuner : paramètres optionnels avec valeurs par défaut
    """
    prix_nuit = 500
    prix_pdj = 50

    total = prix_nuit * nuits
    if petit_dejeuner:
        total += prix_pdj * nuits

    return {
        "client": nom,
        "ville": ville,
        "duree": nuits,
        "pdj": petit_dejeuner,
        "total": total
    }

2. Arguments nommés

Fonction utilisant des arguments nommés :

def calculer_circuit(
        depart="Casablanca",
        destinations=None,
        transport="voiture",
        duree_sejour=None):
    """
    Fonction utilisant des arguments nommés
    """
    if destinations is None:
        destinations = ["Rabat", "Fès"]
    if duree_sejour is None:
        duree_sejour = [2] * len(destinations)

    circuit = {
        "depart": depart,
        "etapes": list(zip(destinations, duree_sejour)),
        "transport": transport
    }
    return circuit

3. Arguments variables (*args, **kwargs)

Fonction avec nombre variable d'arguments :

def organiser_voyage(*villes, **options):
    """
    Fonction avec nombre variable d'arguments
    *villes : tuple des villes à visiter
    **options : dictionnaire des options du voyage
    """
    # Configuration par défaut
    config = {
        "transport": "train",
        "hebergement": "hotel",
        "duree_par_ville": 2,
        "budget_par_jour": 800
    }

    # Mise à jour avec les options fournies
    config.update(options)

    # Calcul du voyage
    planning = []
    for ville in villes:
        etape = {
            "ville": ville,
            "duree": config["duree_par_ville"],
            "budget": config["duree_par_ville"] * config["budget_par_jour"],
            "transport": config["transport"],
            "hebergement": config["hebergement"]
        }
        planning.append(etape)

    return planning

4. Exemple pratique - Système de réservation avancé

class SystemeReservation:
    def __init__(self):
        self.tarifs = {
            "standard": 800,
            "superieure": 1200,
            "suite": 2000
        }
        self.supplements = {
            "vue_mer": 200,
            "balcon": 100,
            "spa": 150
        }

    def calculer_prix(self,
            type_chambre,
            nuits,
            *supplements,
            petit_dejeuner=False,
            demi_pension=False,
            **options_speciales):
        """
        Calcule le prix total avec tous les paramètres possibles
        """
        # Prix de base
        if type_chambre not in self.tarifs:
            raise ValueError(f"Type de chambre inconnu: {type_chambre}")
        total = self.tarifs[type_chambre] * nuits

        # Ajout des suppléments
        for supplement in supplements:
            if supplement in self.supplements:
                total += self.supplements[supplement] * nuits

        # Options de repas
        if petit_dejeuner:
            total += 80 * nuits
        if demi_pension:
            total += 200 * nuits

        # Options spéciales
        for option, valeur in options_speciales.items():
            if option == "reduction_fidelite":
                total *= (1 - valeur)
            elif option == "service_chambre":
                total += 100 * nuits

        return total

5. Exercices pratiques

Exercices à réaliser :

  1. Calculateur de voyage : Créez une fonction qui calcule le coût d'un voyage, utilisez des paramètres par défaut pour les options standard, ajoutez des arguments variables pour les activités, gérez différentes options de transport et d'hébergement.
  2. Gestionnaire d'excursions : Développez une fonction pour organiser des excursions, permettez un nombre variable de destinations, gérez différentes options (guide, repas, transport), calculez les prix selon les options choisies.
  3. Système de réservation restaurant : Créez une fonction de réservation flexible, gérez différentes options (placement, menu, boissons), ajoutez des réductions et suppléments, calculez le prix final selon tous les paramètres.
  4. Projet : Agence de voyage complète : Développez plusieurs fonctions avec paramètres variés, gérez tous types de réservations et options, implémentez un système de fidélité, créez des forfaits personnalisables.
