Les opérateurs arithmétiques et leur priorité

Objectifs d'apprentissage

  1. Maîtriser les opérateurs arithmétiques de base
  2. Comprendre la priorité des opérations
  3. Utiliser les opérateurs composés
  4. Appliquer les opérations dans des cas concrets

1. Opérateurs arithmétiques de base

Exemple avec les prix d'articles dans une boutique marocaine :

# Prix de différents articles dans une boutique marocaine
prix_babouches = 150
prix_tapis = 1200
prix_tajine = 220

# Addition (+)
total = prix_babouches + prix_tapis + prix_tajine
print(f"Total des achats : {total} DH")

# Soustraction (-)
reduction = 100
prix_final = total - reduction
print(f"Prix après réduction : {prix_final} DH")

# Multiplication (*)
nombre_clients = 3
cout_groupe = total * nombre_clients
print(f"Coût pour le groupe : {cout_groupe} DH")

# Division (/)
budget_quotidien = cout_groupe / 7 # Budget par jour pour une semaine
print(f"Budget par jour : {budget_quotidien:.2f} DH")

Résultat de l'exécution :

Total des achats : 1570 DH
Prix après réduction : 1470 DH
Coût pour le groupe : 4710 DH
Budget par jour : 672.86 DH

2. Opérateurs avancés

Division entière (//) et Modulo (%) :

# Division entière (//) personnes = 10
plats_par_table = 3
tables_necessaires = personnes // plats_par_table
print(f"Nombre de tables nécessaires : {tables_necessaires}")

# Modulo (%)
places_restantes = personnes % plats_par_table
print(f"Personnes à placer sur une table incomplète : {places_restantes}")

# Puissance (**)
surface_tapis = 2 ** 2 # Tapis carré de 2 mètres de côté
print(f"Surface du tapis : {surface_tapis} m²")

Résultat de l'exécution :

Nombre de tables nécessaires : 3
Personnes à placer sur une table incomplète : 1
Surface du tapis : 4 m²

3. Priorité des opérations

Ordre de priorité (du plus prioritaire au moins prioritaire) :

  1. ( ) : Parenthèses
  2. ** : Puissance
  3. *, /, //, % : Multiplication, divisions et modulo
  4. +, - : Addition et soustraction

Exemple de calcul avec priorités :

prix_base = 100
reduction_pourcent = 20
taxe_pourcent = 10

# Calcul avec parenthèses pour clarifier l'ordre
prix_final = (prix_base * (100 - reduction_pourcent) / 100) * (1 + taxe_pourcent / 100)
print(f"Prix final avec réduction et taxe : {prix_final:.2f} DH")

Résultat de l'exécution :

Prix final avec réduction et taxe : 88.00 DH

4. Opérateurs composés

Exemples d'utilisation des opérateurs composés :

score_match = 0

# += : Addition et affectation
score_match += 1 # But marqué
print(f"Score après but : {score_match}")

# -= : Soustraction et affectation
budget = 1000
budget -= 150 # Achat de babouches
print(f"Budget restant : {budget} DH")

# *= : Multiplication et affectation
visiteurs = 2
visiteurs *= 3 # Le groupe a triplé
print(f"Nombre final de visiteurs : {visiteurs}")

# /= : Division et affectation
distance = 100
distance /= 2 # Moitié du trajet
print(f"Distance restante : {distance} km")

Résultat de l'exécution :

Score après but : 1
Budget restant : 850 DH
Nombre final de visiteurs : 6
Distance restante : 50.0 km

5. Cas pratiques - Gestion d'un café marocain

Exemple complet utilisant différents opérateurs :

# Prix des articles
the = 8
cafe = 12
jus = 15
gateau = 25

# Calcul d'une commande
nombre_the = 3
nombre_gateaux = 2

# Calcul du total
total_commande = (the * nombre_the) + (gateau * nombre_gateaux)
print(f"\nDétail de la commande :")
print(f"Thé ({nombre_the}x) : {the * nombre_the} DH")
print(f"Gâteaux ({nombre_gateaux}x) : {gateau * nombre_gateaux} DH")
print(f"Total : {total_commande} DH")

# Application d'une réduction de 10% si total > 50 DH
if total_commande > 50:
reduction = total_commande * 0.10
total_commande -= reduction
print(f"Réduction appliquée : {reduction:.2f} DH")
print(f"Total après réduction : {total_commande:.2f} DH")

Résultat de l'exécution :

Détail de la commande :
Thé (3x) : 24 DH
Gâteaux (2x) : 50 DH
Total : 74 DH
Réduction appliquée : 7.40 DH
Total après réduction : 66.60 DH

Opérateurs arithmétiques :

  • + : Addition
  • - : Soustraction
  • * : Multiplication
  • / : Division (résultat float)
  • // : Division entière
  • % : Modulo (reste de division)
  • ** : Puissance

Opérateurs composés :

  • += : Addition et affectation
  • -= : Soustraction et affectation
  • *= : Multiplication et affectation
  • /= : Division et affectation
  • //= : Division entière et affectation
  • %= : Modulo et affectation
  • **= : Puissance et affectation

Bonnes pratiques :

  • Utiliser des parenthèses pour clarifier
  • Éviter les expressions trop complexes
  • Commenter les calculs non évidents
  • Vérifier les types de données
