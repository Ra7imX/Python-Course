Les chaînes de caractères et leurs méthodes essentielles

Objectifs d'apprentissage

  1. Comprendre les différentes façons de créer des chaînes
  2. Maîtriser les méthodes de manipulation de chaînes
  3. Utiliser le formatage de chaînes
  4. Travailler avec les chaînes multilignes

1. Création de chaînes

Il existe plusieurs façons de créer des chaînes en Python :

ville_simple = 'Fès' # Avec guillemets simples
ville_double = "Meknès" # Avec guillemets doubles
description = """La ville de Fès
est la capitale spirituelle
du Maroc.""" # Chaîne multiligne

Points importants :

  • Les guillemets simples et doubles sont équivalents
  • Les triples guillemets permettent de créer des chaînes sur plusieurs lignes
  • Les caractères spéciaux comme \n (nouvelle ligne) sont interprétés dans les chaînes

2. Opérations de base sur les chaînes

Les chaînes de caractères supportent plusieurs opérations :

# Concaténation
nom_complet = "Médina " + "de " + "Fès"
print(nom_complet)

# Répétition
ligne = "-" * 20
print(ligne)

# Longueur
print(f"La chaîne 'Fès' a {len('Fès')} caractères")

Résultat :

Médina de Fès
--------------------
La chaîne 'Fès' a 3 caractères

3. Méthodes de chaînes courantes

Les chaînes possèdent de nombreuses méthodes utiles :

texte = "marrakech la rouge"

# Changement de casse
print(texte.upper()) # MARRAKECH LA ROUGE
print(texte.lower()) # marrakech la rouge
print(texte.title()) # Marrakech La Rouge
print(texte.capitalize()) # Marrakech la rouge

# Recherche et remplacement
adresse = "Avenue Mohammed V, Rabat"
print(adresse.find("Rabat")) # Position de "Rabat"
print(adresse.replace("Rabat", "Salé")) # Remplace "Rabat" par "Salé"

# Vérification du contenu
numero = "123"
texte = "Agadir"
print(numero.isdigit()) # True
print(texte.isalpha()) # True
print(texte.startswith("A")) # True
print(texte.endswith("r")) # True

Méthodes de chaînes courantes :

  • .upper() : Transforme en majuscules
  • .lower() : Transforme en minuscules
  • .title() : Met en majuscule la première lettre de chaque mot
  • .capitalize() : Met en majuscule la première lettre de la chaîne
  • .find() : Trouve la position d'une sous-chaîne
  • .replace() : Remplace une sous-chaîne par une autre
  • .isdigit() : Vérifie si tous les caractères sont des chiffres
  • .isalpha() : Vérifie si tous les caractères sont des lettres

4. Formatage de chaînes

Il existe plusieurs façons de formater des chaînes en Python :

# Ancien style avec %
nom = "Hassan"
age = 25
print("Je m'appelle %s et j'ai %d ans" % (nom, age))

# Méthode format()
ville = "Tanger"
pays = "Maroc"
print("Je suis à {}, {}".format(ville, pays))

# f-strings (recommandé)
temperature = 25
print(f"Il fait {temperature}°C à {ville}")

Résultat :

Je m'appelle Hassan et j'ai 25 ans
Je suis à Tanger, Maroc
Il fait 25°C à Tanger

Avantages des f-strings :

  • Syntaxe simple et lisible
  • Expressions Python directement dans les accolades
  • Performances améliorées
  • Formatage avancé (Ex: f"{nombre:.2f}")

5. Manipulation avancée

Les chaînes permettent des manipulations avancées comme le slicing :

# Slicing (découpage)
mot = "Casablanca"
print(mot[0:4]) # "Casa"
print(mot[4:]) # "blanca"
print(mot[-6:]) # "blanca"

# Strip, split et join
texte = " Bienvenue au Maroc "
print(texte.strip()) # Enlève les espaces
villes = "Rabat,Casablanca,Fès"
liste_villes = villes.split(",") # Création d'une liste
print("-".join(liste_villes)) # Joindre avec des tirets

Résultat :

Casa
blanca
blanca
Bienvenue au Maroc
Rabat-Casablanca-Fès

Notes techniques importantes :

  • Slicing [début:fin:pas] : début inclus, fin exclue
  • strip() : Enlève les espaces au début et à la fin
  • split() : Découpe une chaîne en liste selon un séparateur
  • join() : Joindre les éléments d'une liste en une chaîne

6. Exemples pratiques

Fonction pour formater une adresse marocaine :

def formatter_adresse_marocaine(rue, quartier, ville, code_postal):
    """Formate une adresse marocaine"""
    adresse = f"""
{rue.title()}
{quartier.title()}
{ville.upper()}, {code_postal}
MAROC
""".strip()
    return adresse

# Exemple d'utilisation
adresse = formatter_adresse_marocaine(
    "rue ibn battouta 123",
    "quartier hassan",
    "rabat",
    "10000"
)
print("\nAdresse formatée :")
print(adresse)

Résultat :

Adresse formatée : Rue Ibn Battouta 123 Quartier Hassan RABAT, 10000 MAROC
