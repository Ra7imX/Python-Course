Introduction à l'environnement Python et à l'IDLE

Objectifs d'apprentissage

  1. Comprendre l'environnement Python
  2. Se familiariser avec l'IDLE et l'interpréteur
  3. Distinguer le mode interactif et le mode script
  4. Connaître les outils de base du développement Python

1. L'environnement Python et ses composants

L'environnement Python comprend plusieurs éléments essentiels :

  • L'interpréteur Python :
    • Programme qui lit et exécute le code Python
    • Version actuelle stable : Python 3.x
    • Accessible via la commande 'python' ou 'python3'
  • IDLE (Integrated Development and Learning Environment) :
    • Éditeur de code intégré avec Python
    • Deux modes : Shell (interactif) et Éditeur (fichiers)
    • Fonctionnalités : coloration syntaxique, auto-complétion, débogage
  • Le Shell Python (mode interactif) :
    • Symbole >>> indique que Python attend une commande
    • Exécution immédiate des commandes
    • Idéal pour tester rapidement du code

2. Différence entre mode interactif et mode script

Mode Interactif (Shell) :

>>> 2 + 2
4
>>> print("Salam Maroc!")
Salam Maroc!

Mode Script (Fichier .py) :

  • Le code est écrit dans un fichier
  • Exécuté en une fois avec F5 ou 'Run Module'
  • Permet de sauvegarder et réutiliser le code

3. Premier script dans l'éditeur

print("Bienvenue dans votre premier script Python!")
print("Vous utilisez l'éditeur IDLE")

Résultat après exécution :

Bienvenue dans votre premier script Python!
Vous utilisez l'éditeur IDLE

4. Configuration de l'environnement

Étapes pour bien démarrer :

  1. Installation de Python :
    • Télécharger depuis python.org
    • Cocher "Add Python to PATH" pendant l'installation
    • Vérifier l'installation : python --version
  2. Configuration de IDLE :
    • Options -> Configure IDLE
    • Personnaliser la taille de police
    • Choisir un thème de couleurs
    • Configurer l'indentation (4 espaces recommandés)
  3. Organisation des fichiers :
    • Créer un dossier pour vos projets Python
    • Utiliser des noms explicites pour vos fichiers
    • Toujours ajouter l'extension .py

5. Raccourcis clavier essentiels

Dans IDLE :

  • F5 : Exécuter le script
  • Ctrl+N : Nouveau fichier
  • Ctrl+S : Sauvegarder
  • Ctrl+Z : Annuler
  • Alt+3 : Commenter la ligne
  • Alt+4 : Décommenter la ligne
  • Ctrl+] : Indenter
  • Ctrl+[ : Désindenter

6. Exercices pratiques

Exercices à réaliser :

  1. Configuration de l'environnement :
    • Installez Python sur votre machine
    • Vérifiez la version installée
    • Créez un dossier 'projets_python'
  2. Familiarisation avec IDLE :
    • Ouvrez le shell et faites des calculs simples
    • Créez un nouveau fichier et sauvegardez-le
    • Testez les raccourcis clavier principaux
  3. Premier script :
    • Créez un fichier 'test_env.py'
    • Ajoutez quelques instructions print()
    • Exécutez et observez les résultats
  4. Exploration :
    • Testez la différence entre le shell et l'éditeur
    • Essayez de modifier les préférences de IDLE
    • Créez un script qui affiche des informations sur vous
