Introduction aux modules Python

Objectifs d'apprentissage

  1. Comprendre l'importation de modules
  2. Utiliser les modules standards
  3. Créer ses propres modules
  4. Organiser son code en modules

1. Import de modules standards

Python propose de nombreux modules standards pour faciliter le développement. Voici quelques exemples :

import random
import datetime
import math
from decimal import Decimal

Ces modules permettent de générer des nombres aléatoires, manipuler des dates, effectuer des calculs mathématiques, et travailler avec des nombres décimaux de manière précise.

2. Utilisation des modules pour un système hôtelier

Voici un exemple de classe utilisant plusieurs modules pour gérer un système hôtelier :

class SystemeHotelierAvance:
  def __init__(self):
    self.chambres = {}
    self.reservations = {}

  def generer_numero_reservation(self):
    prefix = datetime.datetime.now().strftime("%Y%m")
    suffix = str(random.randint(1000, 9999))
    return f"RES-{prefix}-{suffix}"

  def calculer_prix_total(self, prix_base, taxes=0.2):
    prix = Decimal(str(prix_base))
    taxe = prix * Decimal(str(taxes))
    return prix + taxe

  def calculer_distance(self, point1, point2):
    return math.sqrt(
      (point2[0] - point1[0])**2 +
      (point2[1] - point1[1])**2
    )

Cette classe utilise les modules datetime, random, math, et decimal pour gérer les réservations, calculer les prix et les distances.

3. Création de modules personnalisés

Vous pouvez créer vos propres modules pour organiser votre code. Par exemple, un module hotel_utils.py pourrait contenir :

def formater_prix(prix):
  return f"{prix:,.2f} DH"

def valider_numero_chambre(numero):
  return len(numero) == 3 and numero.isdigit()

def generer_code_reservation():
  import random
  import string
  chars = string.ascii_uppercase + string.digits
  return ''.join(random.choices(chars, k=8))

Ces fonctions peuvent ensuite être importées dans d'autres fichiers Python.

4. Organisation en modules

Pour organiser un projet Python, vous pouvez structurer vos fichiers comme suit :

projet_hotel/
  ├── __init__.py
  ├── models/
  │ ├── __init__.py
  │ ├── client.py
  │ ├── chambre.py
  │ └── reservation.py
  ├── utils/
  │ ├── __init__.py
  │ ├── validation.py
  │ └── formatage.py
  └── services/
    ├── __init__.py
    ├── reservation_service.py
    └── facturation_service.py

Cette structure permet de séparer les responsabilités et de maintenir un code propre et modulaire.

5. Exemple pratique - Utilisation des modules

Voici un exemple d'utilisation des modules standards :

def demonstration_modules():
  # Utilisation de datetime
  maintenant = datetime.datetime.now()
  print(f"Date actuelle: {maintenant.strftime('%d/%m/%Y')}")

  # Utilisation de random
  numeros_chambres = random.sample(range(100, 150), 5)
  print(f"Chambres sélectionnées: {numeros_chambres}")

  # Utilisation de math
  surface_chambre = math.floor(5.7 * 4.2)
  print(f"Surface de la chambre: {surface_chambre}m²")

  # Utilisation de Decimal pour les calculs précis
  prix = Decimal('199.99')
  taxe = Decimal('0.20')
  total = prix * (1 + taxe)
  print(f"Prix total: {total:.2f} DH")

Ce code montre comment utiliser les modules datetime, random, math, et decimal pour effectuer différentes tâches.
