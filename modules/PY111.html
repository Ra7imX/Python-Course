La boucle for et l'utilisation de range()

Objectifs d'apprentissage

  1. Comprendre la boucle for et son fonctionnement
  2. Maîtriser la fonction range()
  3. Itérer sur différentes séquences
  4. Utiliser des boucles imbriquées

1. Boucle for avec range()

La boucle for est utilisée pour parcourir une séquence d'éléments. Avec range(), elle permet de répéter des actions un nombre précis de fois :

Compteur simple :

for i in range(5):
    print(f"Tour de médina numéro {i+1}")

Résultat :

Tour de médina numéro 1
Tour de médina numéro 2
Tour de médina numéro 3
Tour de médina numéro 4
Tour de médina numéro 5

Range avec début et fin :

for annee in range(2020, 2025):
    print(f"Année touristique : {annee}")

Range avec pas :

for prix in range(100, 501, 100):
    print(f"Forfait disponible : {prix} DH")

2. Itération sur des séquences

La boucle for peut parcourir n'importe quelle séquence : listes, chaînes de caractères, tuples, etc.

Liste des villes :

villes_imperiales = ["Fès", "Meknès", "Marrakech", "Rabat"]
for ville in villes_imperiales:
    print(f"Visite guidée de {ville}")

Chaîne de caractères :

ville = "Agadir"
for lettre in ville:
    print(f"Lettre : {lettre}")

3. Énumération avec enumerate()

La fonction enumerate() permet d'obtenir l'index et la valeur simultanément :

plats_marocains = ["Tajine", "Couscous", "Pastilla", "Harira"]
for index, plat in enumerate(plats_marocains, 1):
    print(f"Plat {index}: {plat}")

Résultat :

Plat 1: Tajine
Plat 2: Couscous
Plat 3: Pastilla
Plat 4: Harira

4. Boucles imbriquées

Les boucles for peuvent être imbriquées pour traiter des structures plus complexes :

jours = ["Lundi", "Mardi", "Mercredi"]
activites = ["Médina", "Musée", "Jardin"]

for jour in jours:
    print(f"\nProgramme du {jour}:")
    for activite in activites:
        print(f"- Visite: {activite}")

Résultat :

Programme du Lundi:
- Visite: Médina
- Visite: Musée
- Visite: Jardin
Programme du Mardi:
- Visite: Médina
- Visite: Musée
- Visite: Jardin
Programme du Mercredi:
- Visite: Médina
- Visite: Musée
- Visite: Jardin

5. Exemple pratique - Planning touristique

Un exemple complet de planning de voyage utilisant les boucles for :

def creer_planning_voyage():
    """
    Crée un planning détaillé pour un voyage au Maroc
    """
    villes = {
        "Marrakech": ["Jemaa el-Fna", "Jardin Majorelle", "Médina"],
        "Fès": ["Médina", "Tanneries", "Médersa"],
        "Chefchaouen": ["Place Outa el Hammam", "Kasbah", "Médina bleue"]
    }
    
    print("\n=== Planning de Voyage au Maroc ===")
    
    # Parcours des villes et leurs attractions
    for num_jour, (ville, sites) in enumerate(villes.items(), 1):
        print(f"\nJour {num_jour} - {ville}")
        print("Programme :")
        
        for num_site, site in enumerate(sites, 1):
            print(f" {num_site}. {site}")
            # Ajout d'informations temporelles
            heure_debut = 9 + (num_site - 1) * 3
            heure_fin = heure_debut + 2
            print(f" {heure_debut}h00 - {heure_fin}h00")

6. Calculateur de budget voyage

Un exemple pratique de calcul de budget avec des boucles for :

def calculer_budget_voyage():
    """
    Calcule le budget total pour un voyage au Maroc
    avec différentes options
    """
    depenses_par_ville = {
        "Casablanca": {
            "hotel": 800,
            "repas": 200,
            "transport": 100
        },
        "Tanger": {
            "hotel": 600,
            "repas": 150,
            "transport": 80
        },
        "Essaouira": {
            "hotel": 500,
            "repas": 150,
            "transport": 70
        }
    }
    
    total_voyage = 0
    print("\n=== Budget détaillé par ville ===")
    
    for ville, depenses in depenses_par_ville.items():
        print(f"\nVille : {ville}")
        total_ville = 0
        
        for categorie, montant in depenses.items():
            print(f" - {categorie.title()}: {montant} DH/jour")
            total_ville += montant
        
        print(f"Total par jour à {ville}: {total_ville} DH")
        total_voyage += total_ville
    
    print(f"\nBudget total par jour: {total_voyage} DH")
    
    # Calcul pour différentes durées
    for jours in [3, 5, 7]:
        print(f"Budget pour {jours} jours: {total_voyage * jours} DH")

7. Notes techniques importantes

Syntaxe de base :

for element in sequence:
    # code à exécuter

Fonction range() :

  • range(stop) : génère des nombres de 0 jusqu'à stop-1
  • range(start, stop) : génère des nombres de start jusqu'à stop-1
  • range(start, stop, step) : génère des nombres de start jusqu'à stop-1 avec un pas défini

Bonnes pratiques :

  • Nommer clairement les variables d'itération
  • Éviter les boucles trop imbriquées
  • Utiliser enumerate() quand l'index est nécessaire
  • Préférer les boucles for aux boucles while quand possible
