Les tuples - Caractéristiques et utilisation

Objectifs d'apprentissage

  1. Comprendre les caractéristiques des tuples
  2. Maîtriser la création et l'utilisation des tuples
  3. Savoir quand utiliser les tuples plutôt que les listes
  4. Appliquer les tuples dans des cas pratiques

1. Création de tuples

# Création directe
coordonnees_koutoubia = (31.6242, -7.9892)
horaires_musee = (9, 00, 17, 30)

# Création par conversion
villes_royales = tuple(["Fès", "Meknès", "Rabat", "Marrakech"])

# Tuple à un élément (noter la virgule)
destination = ("Marrakech",)

2. Accès aux éléments

# Accès par index
latitude = coordonnees_koutoubia[0]
longitude = coordonnees_koutoubia[1]

# Unpacking (déballage)
heure_ouverture, min_ouverture, heure_fermeture, min_fermeture = horaires_musee
print(f"Ouverture : {heure_ouverture}h{min_ouverture}")

3. Utilisation pratique des tuples

class LieuTouristique:
    def __init__(self, nom, coordonnees, horaires):
        self.nom = nom
        self.coordonnees = coordonnees # tuple (latitude, longitude)
        self.horaires = horaires # tuple (heure_ouv, min_ouv, heure_ferm, min_ferm)

    def afficher_informations(self):
        lat, lon = self.coordonnees
        h_ouv, m_ouv, h_ferm, m_ferm = self.horaires

        print(f"\nInformations pour {self.nom}")
        print(f"Coordonnées: {lat}°N, {lon}°E")
        print(f"Horaires: {h_ouv:02d}h{m_ouv:02d} - {h_ferm:02d}h{m_ferm:02d}")

# Création d'instances
jamaa_el_fna = LieuTouristique(
    "Place Jemaa el-Fna",
    (31.6287, -7.9892),
    (8, 0, 23, 0)
)

jardin_majorelle = LieuTouristique(
    "Jardin Majorelle",
    (31.6417, -7.9892),
    (8, 30, 17, 30)
)

4. Exemple pratique - Circuit touristique

def creer_circuit_touristique():
    # Structure: (nom, (latitude, longitude), (durée_visite_heures, prix_entree))
    points_interet = [
        ("Médina de Fès", (34.0333, -5.0000), (4, 0)),
        ("Volubilis", (34.0697, -5.5549), (2, 70)),
        ("Mosquée Hassan II", (33.6067, -7.6327), (1, 120)),
        ("Kasbah des Oudayas", (34.0247, -6.8367), (2, 20))
    ]

    print("\n=== Circuit Touristique au Maroc ===")

    duree_totale = 0
    cout_total = 0

    for site in points_interet:
        nom, coords, details = site
        duree, prix = details
        lat, lon = coords

        print(f"\nSite : {nom}")
        print(f"Localisation : {lat}°N, {lon}°E")
        print(f"Durée de visite : {duree}h")
        print(f"Prix d'entrée : {prix} DH")

        duree_totale += duree
        cout_total += prix

    print(f"\nDurée totale du circuit : {duree_totale}h")
    print(f"Coût total des entrées : {cout_total} DH")

    return duree_totale, cout_total # Retourne un tuple

# Test du circuit
duree, cout = creer_circuit_touristique()

5. Notes techniques importantes

  • Caractéristiques des tuples :
    • Immuables (non modifiables après création)
    • Plus légers que les listes
    • Peuvent être utilisés comme clés de dictionnaire
    • Idéaux pour les données qui ne changent pas
  • Cas d'utilisation :
    • Coordonnées géographiques
    • Horaires
    • Données de configuration
    • Retour multiple de fonctions
  • Avantages :
    • Protection contre la modification accidentelle
    • Performance légèrement meilleure que les listes
    • Code plus clair pour les données fixes
  • Différences avec les listes :
    • Parenthèses () au lieu de crochets []
    • Pas de méthodes de modification
    • Immuabilité garantie
