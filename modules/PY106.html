Saisie utilisateur avec input() et conversion de types

Objectifs d'apprentissage

  1. Maîtriser la fonction input()
  2. Comprendre la conversion de types
  3. Gérer les erreurs de saisie
  4. Créer des interfaces utilisateur simples

1. Base de input()

La fonction input() permet de récupérer des données saisies par l'utilisateur :

# Saisie simple de texte
nom = input("Entrez votre nom : ")
ville = input("Dans quelle ville du Maroc habitez-vous ? ")
print(f"Bonjour {nom} de {ville} !")

Points importants à retenir :

  • La fonction input() affiche le message entre parenthèses
  • Elle met le programme en pause jusqu'à ce que l'utilisateur appuie sur Entrée
  • La valeur retournée est toujours une chaîne de caractères

2. Conversion de types

Comme input() retourne toujours une chaîne, il faut souvent convertir la valeur dans le type souhaité :

# Conversion en entier
age_str = input("Quel âge avez-vous ? ")
age = int(age_str) # Conversion de str vers int

# Conversion en float
taille_str = input("Quelle est votre taille (en mètres) ? ")
taille = float(taille_str) # Conversion de str vers float

Fonctions de conversion courantes :

  • int() : Convertit en nombre entier
  • float() : Convertit en nombre à virgule flottante
  • str() : Convertit en chaîne de caractères
  • bool() : Convertit en booléen

3. Gestion des erreurs de saisie

Si l'utilisateur entre des données non valides, des erreurs peuvent survenir. Utilisez try/except pour les gérer :

try:
    prix_str = input("Prix du tajine (en DH) : ")
    prix = float(prix_str)
    nombre_str = input("Nombre de personnes : ")
    nombre = int(nombre_str)
    
    total = prix * nombre
    print(f"Total à payer : {total:.2f} DH")
except ValueError:
    print("Erreur : Veuillez entrer des nombres valides")

Bonnes pratiques :

  • Toujours vérifier les saisies utilisateur
  • Prévoir des messages d'erreur clairs
  • Gérer spécifiquement les différents types d'erreurs

4. Exemple pratique - Calculateur de voyage

Voici un exemple complet d'interface utilisateur simple pour calculer le coût d'un voyage au Maroc :

def calculer_cout_voyage():
    print("\n=== Calculateur de Voyage au Maroc ===")
    
    # Saisie des informations de base
    try:
        # Nombre de jours
        jours = int(input("Nombre de jours de voyage : "))
        
        # Budget hébergement
        prix_hotel = float(input("Prix de l'hôtel par nuit (DH) : "))
        
        # Budget nourriture
        budget_nourriture = float(input("Budget nourriture par jour (DH) : "))
        
        # Calcul du total
        total_hotel = jours * prix_hotel
        total_nourriture = jours * budget_nourriture
        total_voyage = total_hotel + total_nourriture
        
        # Affichage détaillé
        print("\nRécapitulatif du voyage :")
        print(f"Hébergement : {total_hotel:.2f} DH")
        print(f"Nourriture : {total_nourriture:.2f} DH")
        print(f"Total : {total_voyage:.2f} DH")
        
    except ValueError:
        print("Erreur : Veuillez entrer des nombres valides")

# Exécution du calculateur
calculer_cout_voyage()

Résultat d'exécution :

=== Calculateur de Voyage au Maroc ===
Nombre de jours de voyage : 5
Prix de l'hôtel par nuit (DH) : 350
Budget nourriture par jour (DH) : 200
Récapitulatif du voyage : Hébergement : 1750.00 DH Nourriture : 1000.00 DH Total : 2750.00 DH
