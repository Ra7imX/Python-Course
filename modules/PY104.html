Variables et types de données de base en Python

Objectifs d'apprentissage

  1. Comprendre le concept de variable
  2. Maîtriser les types de données de base
  3. Apprendre les règles de nommage
  4. Pratiquer les opérations sur les variables

1. Déclaration et assignation de variables

Les variables permettent de stocker des données en mémoire :

# Variables simples
nom_equipe = "Wydad" # Type str (chaîne)
annee_creation = 1937 # Type int (entier)
pourcentage_victoires = 75.5 # Type float (décimal)
est_champion = True # Type bool (booléen)
dernier_match = None # Type NoneType

Affichage des variables et leurs types :

Équipe: Wydad (<class 'str'>)
Création: 1937 (<class 'int'>)
Victoires: 75.5% (<class 'float'>)
Champion: True (<class 'bool'>)

2. Types numériques

Entiers (int) :

nombre_joueurs = 11
numero_maillot = 10
age_club = 2025 - annee_creation

Nombres à virgule (float) :

taille_joueur = 1.82
poids_joueur = 75.5
imc = poids_joueur / (taille_joueur ** 2)

3. Chaînes de caractères (str)

Différentes façons de créer des chaînes :

ville = "Casablanca" # Guillemets doubles
stade = 'Mohammed V' # Guillemets simples
adresse = """Bd Mohammed V, # Multiligne
Casablanca, Maroc"""

4. Booléens (bool)

Variables booléennes et comparaisons :

est_professionnel = True
a_gagne_champions_league = False
peut_jouer = age_club > 10 and est_professionnel

5. Conversion de types

Conversion entre différents types :

prix_ticket_str = "150"
prix_ticket_num = int(prix_ticket_str) # str vers int
prix_reduit = float(prix_ticket_num) * 0.8 # int vers float
prix_affiche = str(prix_reduit) # float vers str

Règles de nommage :

  • Commencer par une lettre ou underscore
  • Contenir lettres, chiffres, underscores
  • Sensible à la casse (maVariable ≠ mavariable)
  • Utiliser snake_case en Python (mots_separes_par_underscore)

Types de données de base :

  • str : Chaînes de caractères (texte)
  • int : Nombres entiers
  • float : Nombres décimaux
  • bool : Booléens (True/False)
  • None : Valeur nulle

Bonnes pratiques :

  • Noms descriptifs et significatifs
  • Éviter les noms trop courts ou ambigus
  • Commenter les variables complexes
  • Initialiser les variables avant utilisation

6. Exemples pratiques

Gestion d'un match de football :

equipe_locale = "Raja"
equipe_visiteur = "Wydad"
score_local = 2
score_visiteur = 1
match_termine = True
duree_match = 90.0

# Calcul de statistiques
total_buts = score_local + score_visiteur
difference_buts = abs(score_local - score_visiteur)
match_nul = score_local == score_visiteur
victoire_locale = score_local > score_visiteur

Résultat de l'exécution :

Résultat du match Raja vs Wydad
Score final : 2 - 1
Total des buts : 3
Différence de buts : 1
Match nul : False
Victoire locale : True

    • Créez une fiche d'équipe complète

    • Combinez les chaînes de différentes manières

    • Créez un système de vérification pour un match
    • Utilisez plusieurs conditions
    • Combinez les conditions avec and/or

    • Créez un gestionnaire de tournoi de foot
