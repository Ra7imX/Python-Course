Introduction aux dictionnaires en Python

Objectifs d'apprentissage

  1. Comprendre la structure des dictionnaires
  2. Maîtriser la création et l'accès aux données
  3. Apprendre les opérations de base sur les dictionnaires
  4. Utiliser les dictionnaires dans des cas pratiques

1. Création de dictionnaires

Les dictionnaires en Python sont des collections de paires clé-valeur. Voici quelques exemples de création de dictionnaires :

# Dictionnaire simple hotel = { "nom": "Riad Fès", "etoiles": 5, "ville": "Fès", "piscine": True } # Dictionnaire avec différents types de valeurs restaurant = { "nom": "Dar Marrakech", "specialites": ["Tajine", "Couscous", "Pastilla"], "prix_moyen": 150.50, "livraison": False, "horaires": { "dejeuner": "12h-15h", "diner": "19h-23h" } }

2. Accès aux données

Les dictionnaires permettent un accès facile aux données via des clés. Voici quelques méthodes d'accès :

# Accès direct avec clé print(f"Nom de l'hôtel : {hotel['nom']}") # Accès avec get() (plus sûr) nb_etoiles = hotel.get('etoiles', 'Non classé') print(f"Classification : {nb_etoiles} étoiles") # Accès aux données imbriquées print(f"Horaire dîner : {restaurant['horaires']['diner']}")

3. Exemple pratique - Système de réservation

Voici un exemple pratique d'utilisation des dictionnaires dans un système de réservation :

class SystemeReservation: def __init__(self): # Dictionnaire des riads disponibles self.riads = { "riad1": { "nom": "Riad Royal", "chambres": { "suite": {"prix": 1500, "disponible": True}, "standard": {"prix": 800, "disponible": True}, "familiale": {"prix": 1200, "disponible": False} }, "services": ["Piscine", "Spa", "Restaurant"], "note": 4.8 }, "riad2": { "nom": "Riad Médina", "chambres": { "suite": {"prix": 1200, "disponible": False}, "standard": {"prix": 600, "disponible": True} }, "services": ["Restaurant", "Terrasse"], "note": 4.5 } } def afficher_disponibilites(self): """Affiche les chambres disponibles pour chaque riad""" print("\n=== Disponibilités des Riads ===") for riad_id, info in self.riads.items(): print(f"\n{info['nom']} (Note: {info['note']}/5)") print("Services:", ", ".join(info['services'])) print("Chambres disponibles:") for type_chambre, details in info['chambres'].items(): if details['disponible']: print(f"- {type_chambre.title()}: {details['prix']} DH/nuit") def rechercher_par_budget(self, budget): """Recherche les chambres disponibles dans un budget donné""" print(f"\n=== Chambres disponibles pour {budget} DH ===") for riad_id, info in self.riads.items(): print(f"\nDans {info['nom']}:") chambres_trouvees = False for type_chambre, details in info['chambres'].items(): if details['disponible'] and details['prix'] <= budget: print(f"- {type_chambre.title()}: {details['prix']} DH") chambres_trouvees = True if not chambres_trouvees: print("Aucune chambre dans cette gamme de prix") # Test du système systeme = SystemeReservation() systeme.afficher_disponibilites() systeme.rechercher_par_budget(1000)

4. Exercices pratiques

Exercices à réaliser :

  1. Gestion d'un riad :
    • Créez un dictionnaire pour stocker les informations d'un riad
    • Incluez les chambres, prix, équipements
    • Affichez les informations de manière formatée
    • Ajoutez une fonction de recherche par critères
  2. Guide touristique :
    • Créez un dictionnaire des sites touristiques de votre ville
    • Pour chaque site, stockez : horaires, prix, description, note
    • Ajoutez une fonction qui suggère un circuit selon le temps disponible
    • Implémentez un système de recherche par catégorie
  3. Menu de restaurant :
    • Créez un dictionnaire pour le menu d'un restaurant marocain
    • Organisez par catégorie (entrées, plats, desserts)
    • Incluez prix, description, temps de préparation
    • Ajoutez une fonction pour calculer le total d'une commande
  4. Projet : Application de voyage :
    • Créez une base de données de villes marocaines
    • Pour chaque ville : hôtels, restaurants, activités, transport
    • Implémentez des fonctions de recherche et filtrage
    • Ajoutez un système de recommandation basé sur le budget
