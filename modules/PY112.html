Break et continue : contrôle des boucles

Objectifs d'apprentissage

  1. Comprendre l'utilisation de break
  2. Maîtriser l'utilisation de continue
  3. Combiner les structures de contrôle
  4. Gérer les cas particuliers

1. Utilisation de break

L'instruction break permet de sortir complètement d'une boucle, même si la condition de fin n'est pas satisfaite.

Exemple - Recherche d'un hôtel dans un budget :

hotels = [
    {"nom": "Riad Fès", "prix": 1200},
    {"nom": "Palais Marrakech", "prix": 800},
    {"nom": "Atlas Resort", "prix": 600},
    {"nom": "Médina House", "prix": 400}
]

budget = 700
print("Recherche d'hôtels dans votre budget:")
for hotel in hotels:
    if hotel["prix"] <= budget:
        print(f"Trouvé ! {hotel['nom']} à {hotel['prix']} DH")
        break
    print(f"Vérifiction de {hotel['nom']}...")

Résultat :

Recherche d'hôtels dans votre budget:
Vérifiction de Riad Fès...
Vérifiction de Palais Marrakech...
Trouvé ! Atlas Resort à 600 DH

2. Utilisation de continue

L'instruction continue permet de passer à l'itération suivante de la boucle, en ignorant le reste du code dans la boucle pour l'itération actuelle.

Exemple - Filtrage des activités disponibles :

activites = [
    {"nom": "Visite Médina", "disponible": True},
    {"nom": "Tour en chameau", "disponible": False},
    {"nom": "Cours de cuisine", "disponible": True},
    {"nom": "Spa hammam", "disponible": False}
]

print("Activités disponibles:")
for activite in activites:
    if not activite["disponible"]:
        continue
    print(f"- {activite['nom']}")

Résultat :

Activités disponibles:
- Visite Médina
- Cours de cuisine

3. Combinaison de break et continue

Les instructions break et continue peuvent être combinées pour créer des algorithmes plus complexes.

Exemple - Recherche de voyage :

def rechercher_voyage():
    """
    Recherche un voyage selon des critères spécifiques
    """
    voyages = [
        {"destination": "Marrakech", "prix": 3000, "disponible": True},
        {"destination": "Agadir", "prix": 2500, "disponible": False},
        {"destination": "Tanger", "prix": 2000, "disponible": True},
        {"destination": "Essaouira", "prix": 1800, "disponible": True}
    ]
    
    budget_max = 2200
    print(f"Recherche de voyages à moins de {budget_max} DH:")
    
    for voyage in voyages:
        # Sauter les voyages non disponibles
        if not voyage["disponible"]:
            print(f"Voyage à {voyage['destination']} non disponible, recherche suivante...")
            continue
            
        # Vérifier le budget
        if voyage["prix"] <= budget_max:
            print(f"Voyage trouvé : {voyage['destination']} à {voyage['prix']} DH")
            break
        
        print(f"Voyage à {voyage['destination']} trop cher, recherche suivante...")
    else:
        print("Aucun voyage trouvé dans votre budget")

Résultat :

Recherche de voyages à moins de 2200 DH:
Voyage à Marrakech trop cher, recherche suivante...
Voyage à Agadir non disponible, recherche suivante...
Voyage trouvé : Tanger à 2000 DH

4. Exemple pratique - Système de réservation

Voici un exemple plus complexe combinant les structures de contrôle :

def systeme_reservation():
    """
    Système de réservation pour un restaurant marocain
    """
    places_disponibles = 20
    reservations = []
    
    while True:
        print("=== Système de Réservation ===")
        print(f"Places disponibles : {places_disponibles}")
        print("1. Nouvelle réservation")
        print("2. Annuler réservation")
        print("3. Voir réservations")
        print("4. Quitter")
        
        choix = input("Votre choix : ")
        
        if choix == "1":
            nombre_personnes = int(input("Nombre de personnes : "))
            if nombre_personnes <= 0:
                print("Nombre invalide")
                continue
                
            if nombre_personnes > places_disponibles:
                print("Désolé, pas assez de places")
                continue
                
            nom = input("Nom de la réservation : ")
            reservations.append({"nom": nom, "personnes": nombre_personnes})
            places_disponibles -= nombre_personnes
            
        elif choix == "2":
            # Code pour annuler une réservation
            # ... (code omis pour plus de clarté)
            
        elif choix == "3":
            # Code pour afficher les réservations
            # ... (code omis pour plus de clarté)
            
        elif choix == "4":
            print("Au revoir!")
            break
            
        else:
            print("Choix invalide")
            continue

5. Notes techniques importantes

  • Break :
    • Sort complètement de la boucle
    • Utile pour arrêter une recherche
    • Souvent utilisé avec while True
  • Continue :
    • Passe à l'itération suivante
    • Utile pour sauter des cas particuliers
    • Évite les conditions imbriquées
  • Bonnes pratiques :
    • Utiliser break avec parcimonie
    • Préférer des conditions claires
    • Documenter l'utilisation de break/continue
