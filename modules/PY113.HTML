Les listes - Création et manipulation de base

Objectifs d'apprentissage

  1. Comprendre la création et l'initialisation des listes
  2. Maîtriser l'accès et la modification des éléments
  3. Apprendre les opérations de base sur les listes
  4. Utiliser le slicing sur les listes

1. Création de listes

Différentes façons de créer des listes :

villes_maroc = ["Casablanca", "Rabat", "Fès", "Marrakech"]
notes_voyage = [18, 15, 17, 19, 16]
liste_mixte = ["Tanger", 35, True, 450.75]
liste_vide = []

# Liste par compréhension
temperatures = [temp for temp in range(20, 31, 2)] # [20, 22, 24, 26, 28, 30]

2. Accès aux éléments

Accès par index :

premiere_ville = villes_maroc[0] # Premier élément
derniere_ville = villes_maroc[-1] # Dernier élément

print(f"Première ville : {premiere_ville}")
print(f"Dernière ville : {derniere_ville}")

3. Modification d'éléments

Modification simple :

villes_maroc[1] = "Meknès"
print(f"Liste mise à jour : {villes_maroc}")

# Modification multiple
villes_maroc[1:3] = ["Tétouan", "Tanger"]
print(f"Après modification multiple : {villes_maroc}")

4. Slicing (découpage)

Extraire une partie de la liste :

plats_marocains = ["Tajine", "Couscous", "Pastilla", "Harira", "Tanjia"]

premiers_plats = plats_marocains[0:2] # ["Tajine", "Couscous"]
derniers_plats = plats_marocains[-2:] # ["Harira", "Tanjia"]
plats_milieu = plats_marocains[1:4] # ["Couscous", "Pastilla", "Harira"]

# Slicing avec pas
plats_alternes = plats_marocains[::2] # ["Tajine", "Pastilla", "Tanjia"]

5. Opérations sur les listes

Concaténation et répétition :

restaurants_casa = ["Rest1", "Rest2"]
restaurants_rabat = ["Rest3", "Rest4"]
tous_restaurants = restaurants_casa + restaurants_rabat

# Répétition
ligne_separation = ["-"] * 20

# Vérification d'appartenance
if "Tajine" in plats_marocains:
    print("Le tajine est un plat marocain")

6. Exemple pratique - Gestion d'un circuit touristique

def gerer_circuit_touristique():
    # Initialisation du circuit
    circuit = {
        "villes": ["Marrakech", "Essaouira", "Agadir"],
        "durees_sejour": [3, 2, 2],
        "hotels": ["Riad Marrakech", "Dar Essaouira", "Resort Agadir"],
        "activites": [
            ["Médina", "Jardin Majorelle", "Jemaa el-Fna"],
            ["Port", "Médina", "Plage"],
            ["Plage", "Souk", "Kasbah"]
        ]
    }

    # Affichage du circuit
    print("\n=== Circuit Touristique au Maroc ===")

    for i in range(len(circuit["villes"])):
        ville = circuit["villes"][i]
        duree = circuit["durees_sejour"][i]
        hotel = circuit["hotels"][i]
        activites = circuit["activites"][i]

        print(f"\nÉtape {i+1}: {ville}")
        print(f"Durée: {duree} jours")
        print(f"Hébergement: {hotel}")
        print("Activités:")
        for j, activite in enumerate(activites, 1):
            print(f" {j}. {activite}")

# Test du circuit
gerer_circuit_touristique()

7. Notes techniques importantes

  • Caractéristiques des listes :
    • Ordonnées (ordre préservé)
    • Mutables (modifiables)
    • Peuvent contenir des doublons
    • Peuvent contenir différents types
  • Indexation :
    • Commence à 0
    • Index négatifs acceptés
    • Erreur IndexError si hors limites
  • Slicing :
    • liste[début:fin:pas]
    • début inclus
    • fin exclue
    • pas optionnel
  • Bonnes pratiques :
    • Utiliser des noms descriptifs
    • Garder les listes homogènes si possible
    • Commenter les opérations complexes
